［Python］安裝pyenv - Python各版本環境切換(for Mac OS)

Written by Syuan Shih with ♥ on February 16, 2019 in Data Science

使用pyenv這個套件即可下指令切換本機Python的執行版本環境。

Step1：使用Homebrew安裝pyenv

$ brew update
$ brew install pyenv

Step2：修改設定檔 - 設定環境變數

  • bash：修改 ~/.bashrc
  • zsh：修改 ~/.zshrc

新增下方兩行：

export PATH="$HOME/.pyenv/bin:$PATH"
eval "$(pyenv init -)"

最後重新更新 ~/.bashrc or ~/.zshrc 或重新啟動terminal

$ source ~/.bashrc
$ source ~/.zshrc

Step3：依需要安裝python各版本

  • 檢視可以安裝的版本(純版本號為官方版本，其他為衍生版本)
$ pyenv install --list
  • 安裝需要的版本
$ pyenv install 3.7.2
$ pyenv install 2.7.10
  • 檢視系統已安裝的版本
$ pyenv versions

# output(*表示當前系統使用的版本)
* system (set by [HOME DIR]/.pyenv/version)
  2.7.10
  3.7.2
  • 確定安裝完成後，更新資料庫
$ pyenv rehash
  • 切換python版本
$ pyenv global 3.7.2
$ pyenv global 2.7.10
  • 檢視目前系統python的版本
# 第一種方式(看*位置是否在設定的版本號前方)
$ pyenv versions

# 第二種方式(使用python指令查看)
$ python --version

# 第三種方式(直接進入python指令環境)
$ python
# 進入後會寫目前系統版本號
>>> 可輸入指令
>>> exit() #退出python指令環境
